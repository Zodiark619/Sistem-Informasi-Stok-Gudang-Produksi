Di command prompt akses lokasi folder tempat diletakkan app inventory, kemudian lakukan perintah berikut ini di command prompt.
venv\Scripts\activate
cd mysite
python manage.py runserver

Kemudian di web browser akses link http://127.0.0.1:8000/inventory/ . Link ini adalah halaman utama dari Sistem Informasi Stok Gudang Produksi.

Jika terdapat halaman lagin maka ketikkan usernya admin dan passwordnya adminadmin. Jika berhasil maka akses ke halaman admin berhasil.
Halaman admin diakses dengan cara akses link http://127.0.0.1:8000/admin/inventory/ .
